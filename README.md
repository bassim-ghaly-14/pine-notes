# 🍍 Pine Notes

A modern and elegant Notes App built with Vanilla JavaScript.

## ✨ Features

- Add Notes
- Delete Notes
- Pin Important Notes
- Search Notes
- Filter By Category
- Dark / Light Mode
- Responsive Design
- Local Storage Persistence
- Apple-Inspired UI

---

## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Local Storage API

---

## 🚀 Getting Started

1. Download the project
2. Open `index.html`
3. Start writing notes instantly
